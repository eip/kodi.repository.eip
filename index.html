eip's Kodi addons repository

Repository for eip's Kodi® addons.

Requirements

  • XBMC v13 (Gotham) or Kodi v14 (Helix) or newer version.

Installation

Download the repository.eip-1.0.1.zip file and install the addon (Settings → Add-ons → Install from zip file → Navigate to the ZIP file).

See Kodi Wiki for more details.
